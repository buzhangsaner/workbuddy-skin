# 人民智造主题设计

## 目标

新增第 9 套预设主题“人民智造”，复现参考图的宣纸米白、朱红、鎏金、山河与放射光氛围，同时完整保留 WorkBuddy 现有页面结构、控件数量、交互和布局。主题必须继续使用现有声明式目录、Renderer 动态加载与安全图片校验，不新增主题专属 DOM 或 JavaScript 分支。

## 视觉方案

主题背景是一张不含文字、Logo、菜单、按钮、卡片或输入框的纯场景图：红金历史画卷质感，远山、长城、红旗、金色晨光和一位居右的历史人物肖像，为左侧原生内容保留可读空间。页面色板采用朱红 `#b51f1a`、鎏金 `#d6a23a`、宣纸米白 `#fbf5e9` 和墨褐 `#2b2018`。原生侧栏、卡片、输入框和对话区域只通过颜色、边框、阴影、透明度和字体风格形成红金宣纸效果。

## 结构约束与验证

新增 `themes/people-intelligence/theme.json` 与一张 `hero-clean-v1` 图片，并在 `themes/catalog.json` 注册。Renderer 不得出现 `people-intelligence` 特判；主题专属 CSS 不得使用 `display`、`position`、方向偏移、宽高、网格、间距、排序或 `transform` 等布局属性。测试先锁定目录顺序、主题合法性、纯背景资产和无结构属性约束；实现后运行完整测试，再在真实 WorkBuddy 中点击主题按钮，核对背景、文字对比度、侧栏、卡片、输入区、项目页和对话页。
